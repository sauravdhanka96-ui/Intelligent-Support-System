# Intelligent Support Ticketing & Triage System

## Overview
This system automates the triage of IT support tickets using keyword-based prioritization. It helps support teams reduce resolution time by ensuring critical issues are flagged immediately.

## Features
* **Auto-Triage Logic**: Automatically assigns High/Medium/Low priority based on ticket content.
* **Database Management**: Structured SQL tables for Tickets, Users, and Knowledge Base articles.
* **SLA Tracking**: Tracks ticket status and creation timestamps.

## Technologies Used
* **C++**: Used for memory-efficient triage logic.
* **SQL**: Used for data integrity and storage.
