# Intelligent Support Ticketing & Triage System

A C++ system that automatically classifies incoming support tickets by priority (High / Medium / Low) using keyword-based triage logic — reducing manual sorting time for helpdesk teams.

---

## What It Does

- **Auto-triage** — parses ticket content and assigns priority using rule- and pattern-based classification
- **3-tier prioritization** — High, Medium, Low based on keyword matching against ticket content
- **SLA tracking** — tracks ticket status and creation timestamps
- **Structured storage schema** — SQL schema for Tickets, Users, and Knowledge Base articles

---

## Tech Stack

| Layer | Technology |
|---|---|
| Language | C++ |
| Classification | Keyword & rule-based logic |
| Storage Schema | SQL |
